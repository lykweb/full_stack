# 树

## 前序遍历

### 什么是前序遍历

> 前序遍历（Preorder Traversal）是二叉树遍历的一种方式，是一种深度优先遍历（Depth-First Traversal）的方法。在前序遍历中，树的节点按照如下顺序被访问：根节点 -> 左子树 -> 右子树。

例如，假设有如下的二叉树：

```typescript
        1
       / \ 
      2   3
     / \ / \
    4  5 6  7
```

前序遍历的结果是 [1, 2, 4, 5, 3, 6, 7]。

### 前序遍历的使用场景

前序遍历通常用在二叉树中按照根节点优先的顺序进行访问，可以用于从根节点开始对整棵树进行处理，或者用于复制一颗二叉树，或者用于输出表达式树的前缀表达式等应用场景。在实现二叉树的前序遍历时，可以使用递归或者迭代的方式。

### leetcode 144 二叉树的前序遍历

> 利用栈进行二叉树前序遍历[leetcode题号144](https://leetcode.cn/problems/binary-tree-preorder-traversal/description/)，请在小打卡准确描述实现代码时间复杂度和空间复杂度.

``` javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var preorderTraversal = function(root) {
    // 根 -> 左- > 右
    const res = [];
    const stack = [];

    // 根节点入栈
    if (root) {
        stack.push(root);
    }

    // 遍历
    while (stack.length) {
        // 弹出栈顶元素
        const node = stack.pop();
        res.push(node.val);

        // 右节点先入栈，左节点后入栈，弹出栈顶元素按照根左右顺序
        if (node.right) {
            stack.push(node.right);
        }
        if (node.left) {
            stack.push(node.left);
        }
    }

    return res;
};
```
